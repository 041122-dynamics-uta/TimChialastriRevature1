WILCARDS NEEDED: % WC 1 TO SELECT CHARCTER(S) VALUE IN DESIRED POSITION               MODULUS B4 TO GRAB CHARACTER AT END, MODULUS AFTER CHARACTER FOR FIRST CHARACTER
LIKE: NEEDED TO ASK FOR LIKE VALUES
NOT: BECAUSE THIS PROBLEM IS ASKING FOR VALUES THAT DO not START WITH VOWELS

SELECT DISTICT CITY
FROM STATION
WHERE not (city like 'A%' or city like 'E%' or city like 'I%' or city like 'O%' or city like 'U%');
